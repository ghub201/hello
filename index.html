Project focus

A balanced model for digital communication security

The prototype presents a research-based framework that avoids relying only on a centralized key server, a public blockchain, or a private blockchain.

Security

Protects sensitive key operations through private validation, access policies, and controlled lifecycle management.

Trust

Anchors selected proof records to a public blockchain so key events can be independently verified without revealing secrets.

Performance

Moves frequent key lifecycle actions into a private blockchain layer where processing is faster and more controlled.

Hybrid design

Public verification and private execution working together

The proposed framework separates operational key management from public audit proof. This separation supports confidentiality, accountability, and performance in the same model.
