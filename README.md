# HybridKey Shield

A static GitHub Pages-ready website prototype for a research project on a Hybrid Blockchain-Based Key Management Framework.

## Files
- `index.html` - Home page
- `framework.html` - Proposed conceptual framework
- `lifecycle.html` - Key lifecycle page
- `security.html` - Security comparison page
- `demo.html` - Interactive key management workflow demo
- `references.html` - Selected references
- `styles.css` - Full responsive design
- `script.js` - Navigation and demo interactivity

## How to host on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. Open repository Settings.
4. Go to Pages.
5. Select the `main` branch and root folder.
6. Save and wait for GitHub to publish the site.

No external dependencies are required.
